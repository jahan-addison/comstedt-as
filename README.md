# comstedt-as

A fork of the macro assembler written in ANSI-C for the Dreamcast's Visual Memory System by Marcus Comstedt.

#### Features

* Uses a more modern and portable build process via cmake
* Compiles for windows, BSD (macos), and linux platforms

## Build

```bash
cmake -B build
cmake --build build
```
